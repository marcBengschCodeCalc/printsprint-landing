printsprint is a private B2B fulfillment tool for Etsy sellers. Etsy customers place and pay for their orders directly on Etsy as usual. This app enables sellers to automatically forward those orders to our backend system for production and shipping.

How It Works

  • Sellers log in with their Etsy account using OAuth 2.0
  • printsprint accesses order and listing data with permission
  • Our system fulfills the order (production + shipping)
  • Sellers are billed through Stripe for the service

Etsy API Usage

The app uses the following Etsy API endpoints with seller consent:

  • /shops/:shop_id/receipts ‚Äì to fetch recent orders
  • /listings/:listing_id ‚Äì to get product details
  • /users/:user_id ‚Äì for identity verification

All data access is read-only and strictly limited to the authenticated shop owner's data. We fully comply with Etsy's API Terms of Use.
